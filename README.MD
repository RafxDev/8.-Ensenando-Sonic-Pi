# 📖 Acerca del proyecto

**Enseñando Sonic Pi** es un curso gratuito y de código abierto cuyo objetivo es enseñar a utilizar **Sonic Pi** desde los conceptos más básicos hasta técnicas avanzadas de programación musical.

El contenido está organizado como un libro, permitiendo avanzar paso a paso sin asumir conocimientos previos sobre programación musical, teoría musical o el lenguaje Ruby.

Este repositorio busca convertirse en un recurso de referencia para la comunidad hispanohablante interesada en el **Live Coding** y la creación musical mediante código.

---

# 🎯 Objetivos

Al finalizar este curso podrás:

- Comprender el funcionamiento de Sonic Pi.
- Aprender los fundamentos de Ruby aplicados a Sonic Pi.
- Crear melodías y ritmos utilizando código.
- Trabajar con sintetizadores y efectos.
- Construir composiciones completas.
- Aprender técnicas de Live Coding.
- Desarrollar proyectos musicales propios.

---
